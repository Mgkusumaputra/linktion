## Features 🧱
- 🏆 Unlimited Linktree
- 🔐 Self-hostable with your own domain
- ⚙ Using Notion as a powerful CMS
- 📊 Linktree click count statistics
- 🛠 No hard coding, all configuration from Notion
- 🎨 Custom themes
